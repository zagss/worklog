# LaFeng-App
this is a app and code by reactjs

### 下载
 git clone https://github.com/zagss/LaFengApp.git   
 cd  LaFeng-App   
 npm install    
 npm install webpack -g

### 运行
  npm start

### 发行
  webpack -p





### 监听命令
  webpack --progress --colors -watch


### 开发服务器
  npm install webpack-dev-server -g  

## 启动
  webpack-dev-server --progress --colors
